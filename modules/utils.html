Documentation for utils.py¶

General string parsing and manipulation functions

str2d.utils.chunk(s: str, n: int) → map[source]¶

Split a long string into n size pieces

Parameters:
  • s – Thing to be chunked
  • n – Size of each chunk
Returns:

map object that is an iterable of strings

 
>>> from str2d import utils
... tuple(utils.chunk('abcdefghij', 2))
('ab', 'cd', 'ef', 'gh', 'ij')
str2d.utils.mask(inp: str, msk: str, char: str = ' ', replace: str = None, invert: bool = False) → str[source]¶

Mask a string

Parameters:
  • inp – String to mask
  • msk – String to use as mask
  • char – Character within msk that will do the masking
  • replace – What to replace char with. If None will use characters from msk
  • invert – We could invert the mask
Returns:

str

 
>>> from str2d import utils
... utils.mask('abcdefghij', '-' * 5 + ' ' * 5, invert=True)
'abcde     '

>>> from str2d import utils
... utils.mask('abcdefghij', '-' * 5 + ' ' * 5, invert=False)
'-----fghij'
